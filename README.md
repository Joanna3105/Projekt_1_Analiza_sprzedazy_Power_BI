# Projekt 1: Analiza sprzedaży w Power BI

## Opis projektu

Projekt obejmuje interaktywny dashboard analizujący:
- sprzedaż,
- rentowność,
- klientów,
- produkty,
- transport.

## Wykorzystane narzędnia

- Power BI,
- Power Query,
- DAX,
- modelowanie danych.

## Zakres prac

- przygotowanie danych w Power Query i stworzenie tabeli Kalendarz,
- model danych i relacje, 
- utworzenie miar DAX,
- projekt interaktywnego dashboardu,
- wizualizacja danych,
- wyciągnięcie wniosków na stan obecny dashboardu.

## Zawartość repozytorium

- dashboard.pbix
- dane.xls
- zrzuty ekranu dashboardu.pdf
Do projektu dołączyłam również wnioski, pokazując szerzej swoją interpretację otrzymanych wyników - są one aktualne dla danego zestawu danych.
